Minha biblioteca
Site apresentado para Disciplina de Desenvolvimento Web 1
Curso de Tads UFPR


## Tema

O projeto final desta disciplina de Sistemas de Desenvolvimento Web I é uma Biblioteca Virtual. Optamos pela criação de um site que abrange todas as tecnologias abordadas durante o curso.

## Objetivo do Projeto

A finalidade deste projeto é criar um site funcional com interface amigável e intuitiva, fazendo uso das seguintes tecnologias: HTML, CSS, Javascript, MySQL, PHP, entre outras.

## Delimitação do Problema

O grande desafio inicial foi a definição do projeto, partindo de uma premissa que englobe a área de educação e esteja alinhada com os princípios da equipe. Isso inclui estabelecer requisitos funcionais e não funcionais, escolher as tecnologias a serem usadas e entender como implementá-las, além de gerenciar o tempo para construir um sistema funcional e atrativo para diferentes usuários.

## Justificativa do Tema

A ideia surgiu do uso cotidiano de sites de livros. Todos os membros da equipe são estudantes que frequentemente pesquisam livros na internet. Ao analisar as opções de sites existentes, identificamos novas funcionalidades que poderiam ser implementadas em um projeto deste nicho.

## Método de Trabalho

A organização foi realizada utilizando a metodologia em Cascata, que envolve uma sequência de fases claramente definidas, onde cada fase só avança após a aprovação da fase anterior. As atividades foram organizadas em blocos e posteriormente em tarefas.
